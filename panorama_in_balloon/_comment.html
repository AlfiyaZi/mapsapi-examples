Данный пример демонстрирует, как отобразить панораму в балуне.

В примере на карту добавлены две метки, при клике на которые в балуне открывается панорама для текущей точки.

Чтобы открыть панораму в балуне, необходимо задать для него макет с помощью фабрики . В HTML-шаблоне, на основе которого будет строиться макет, нужно определить блочный контейнер - в нем будет размещена будущая панорама. Обратите внимание, что для этого контейнера должны быть заданы размеры (в противном случае панорама не отобразится).

В данном примере размеры контейнера с панорамой определяются динамически, в зависимости от того, будет ли найдена панорама для данной точки или нет. Проверка на наличие панорамы выполняется при построении макета, при вызове функции build. Если панорама была найдена, то для контейнера задаются размеры 300х200 пикселей (эти значения записываются в поле data.Manager.state.size). После этого в этот контейнер добавляется плеер панорам (см. в примере функцию _createPanoramaPlayer). Если для данной точки панорамы нет, то для контейнера будут заданы меньшие размеры (100x50 пикселей).

Обратите внимание, что каждый раз при изменении размеров контейнера макет балуна будет перестраиваться. Следите за тем, чтобы после изменения размеров контейнера не было повторного запроса данных для одной и той же панорамы.
