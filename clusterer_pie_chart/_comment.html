Pie chart layout . Available in the layout storage with the key 'default#pieChart'. You can use this layout as a tool for visualizing any data, or in combination with other visual API components such as placemarks, the clusterer, and the object manager.

The layout can be based on statistical data, which should be an array of JSON objects with the weight and color fields (the weight and color of a section), or a function that returns such an array (see the Placemark example). If properties.data is omitted, the layout iterates through all the geo objects, determines the value of the iconColor option for each object, and creates the diagram based on this data.
