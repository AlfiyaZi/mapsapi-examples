API Яндекс.Карт может использоваться для отображения на сайте панорам.

Панорама реализуется объектом Panorama. Этот объект содержит необходимую информацию о панораме: ее название, географические координаты точки, к которой привязана панорама, связи со смежными панорамами и др.

Для добавления панорамы на страницу предназначен объект panorama.Player. Он формирует запросы за тайлами и создает на странице элемент <canvas> для отображения панорамы.

Перед тем как создать плеер, необходимо получить объект панорамы с помощью функции panorama.locate.
