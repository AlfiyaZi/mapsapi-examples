Placemark images can be set using CSS sprites.

To set the correct image from a sprite for a placemark, define the following options for this placemark:

  • iconImageCliptRect — The position of the image in the sprite. It is defined by the pixel coordinates of the upper-left and lower-right corners of the rectangle that the image is placed in.
  • iconImageHref — URL of the spirte.
  • iconImageSize — Image dimensions.
  • iconImageOffset — Offset of the icon relative to the anchor point.
  • iconLayout — Icon layout. For options, the 'default#image' value must be set.

For more information about using CSS sprites, refer to the documentation.
