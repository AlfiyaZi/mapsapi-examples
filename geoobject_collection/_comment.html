В API реализовано два вида коллекций геообъектов: упорядоченные () и неупорядоченные (). Первые реализованы на основе массивов, вторые — на основе двусвязных списков.

Геообъект может принадлежать только одной коллекции. При добавлении геообъекта, принадлежащего одной коллекции, в другую коллекцию, он будет удален из первой.

Поиск элементов в упорядоченных коллекциях производится быстрее, чем в неупорядоченных. Однако, удаление элементов из упорядоченных коллекций - дорогостоящая с точки зрения потребления ресурсов операция и производится медленее, чем удаление элементов из неупорядоченных коллекций.
