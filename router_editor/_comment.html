Ссылка на редактор маршрута находится в поле editor экземпляра класса router.Route, который будет передан в качестве параметра объекту после того, как выполнится построение маршрута.

Включать (выключать) редактор можно только внутри функции, которая выполнится, когда от сервера придет результат. 

ymaps.route(...) // возвращает объект util.Promise
    .then(
        function (route) {
            route.editor.start();
            ...
        })
    )
