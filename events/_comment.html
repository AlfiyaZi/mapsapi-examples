В API Яндекс.Карт реализован собственный механизм работы с событиями, расширяющий стандартные средства управления событиями JavaScript.

Технология работы с событиями API стандартная: подписываемся на событие и назначаем функцию-обработчик, которая вызывается при его возникновении.

При назначении обработчика события можно указать не только функцию-обработчик, но и контекст, в котором она будет выполняться.
