The API allows you to create your own custom maps.

For creating a custom map, first of all you need to set the coordinate system that the map will be projected on. The API has built-in support for two coordinate systems: Cartesian and geographical, which are implemented by the and classes, respectively. It is also possible to create a custom coordinate system.

In addition, creating a custom map requires making your own custom layer and adding it to the map layer storage. To create a layer, use the class. When adding the layer to the layer storage, specify the layer constructor, not an instance of the Layer class.
