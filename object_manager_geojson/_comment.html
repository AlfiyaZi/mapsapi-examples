The example shows how to use to display data on the map after exporting it from the Map Constructor. The JQuery.getJSON() function loads the data. As soon as data is loaded, we set properties for it (such as the balloon content) and add it to the object manager.

For more information about exporting, refer to the documentation.

Note that the object coordinates are set in the order "longitude, latitude". Since the API uses "latitude, longitude" by default, the "coordorder" parameter with the "lotlang" value must be passed when enabling the API. For more information, see the section Enabling the API.
