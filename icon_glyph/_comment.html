API позволяет добавлять на карту метки, содержащие глиф-иконки (Glyph).

Для создания метки с глиф-иконкой необходимо указать значение опции "preset" с префиксом "glyph" (например, "islands#glyphCircleIcon"). Через опцию "iconGlyph" нужно задать название глиф-иконки. Для задания цвета иконки предназначена опция "iconGlyphColor".

Обратите внимание, что для работы данного примера нужно подключить Bootstrap.
